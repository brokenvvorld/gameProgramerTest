# gameProgramerTest

一个为**游戏程序员**设计的前端测评项目，题目同时覆盖客户端与服务器工程实践，也包含稳定性、质量与协作倾向。

> 说明：本项目题目与视觉均为原创实现，仅借鉴“答题-出结果”的交互流程，不复用第三方站点视觉与文案。

## 运行

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

## 部署

项目目标部署地址：

- https://brokenvvorld.github.io/gameProgramerTest/
